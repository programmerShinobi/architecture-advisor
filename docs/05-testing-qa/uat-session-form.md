## SUS-10 (post-session questionnaire)

Ask the participant to rate **1 (Sangat tidak setuju / Strongly disagree) … 5 (Sangat setuju /
Strongly agree)** — quickly, first instinct, every item.

| # | Pernyataan (ID) | Statement (EN) | 1–5 |
|---|---|---|---|
| 1 | Saya rasa saya akan sering menggunakan aplikasi ini | I think I would like to use this system frequently | |
| 2 | Saya merasa aplikasi ini terlalu rumit | I found the system unnecessarily complex | |
| 3 | Saya merasa aplikasi ini mudah digunakan | I thought the system was easy to use | |
| 4 | Saya rasa saya butuh bantuan orang teknis untuk memakainya | I think I would need the support of a technical person to use this system | |
| 5 | Saya merasa fitur-fiturnya terpadu dengan baik | I found the various functions well integrated | |
| 6 | Saya merasa terlalu banyak yang tidak konsisten | I thought there was too much inconsistency | |
| 7 | Saya rasa kebanyakan orang akan cepat bisa memakainya | I imagine most people would learn this system very quickly | |
| 8 | Saya merasa aplikasi ini canggung/merepotkan dipakai | I found the system very cumbersome to use | |
| 9 | Saya merasa percaya diri menggunakan aplikasi ini | I felt very confident using the system | |
| 10 | Saya perlu belajar banyak hal dulu sebelum bisa memakainya | I needed to learn a lot before I could get going | |

**Scoring** (Brooke, 1996): odd items → (rating − 1); even items → (5 − rating); sum × 2.5 = **0–100**.

**SUS score: ___ / 100** *(SRS NFR-USE-1 target: mean ≥ 70 across participants)*
